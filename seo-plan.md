# 马来西亚新山借贷站（单页）SEO 与内容规范

## 目标与定位

- 目标地区：马来西亚（重点新山 Johor）
- 用户意图：合法合规、快速审批（1-2 小时）、无需定金押金、免费申请、低利率、流程简单方便
- 页面形式：单页（One Page），以强 CTA 与表单转化为核心

## 关键字策略（Keyword Strategy）

1. **主关键词**：新山贷款、马来西亚贷款、合法贷款、快速审批
2. **长尾关键词**：新山合法贷款申请、1-2 小时审批贷款、无需定金押金贷款、Johor personal loan
3. **选词方法**：
   - 结合用户需求与搜索意图
   - 评估流量潜力、竞争程度（KGR、Low DR）
   - 强相关性与商业价值优先
4. **页面部署**：
   - Title / Meta / H1 / H2-H3 统一围绕「合法 + 快速 + 新山」
   - 首段落 120-160 字内自然出现主关键词

## 内容大纲与结构（参考 Top10 SERP）

建议结构（已在 `index.html` 体现）：

- Hero：合法合规、快速审批、无需定金、免费申请
- 优势：合法合规、1-2 小时审批、低利率、流程简单
- 方案：个人贷款 / 企业资金 / 分期 / 债务整合
- 申请流程：3 步（申请 → 审核 → 确认）
- 代表性示例：APR 范围 + 月供示例
- FAQ：PAA 常见问题覆盖
- 评价：增强信任
- 表单：强 CTA 引导转化

## 内容优化细节（On-page SEO）

- **Title**：包含“新山”“合法贷款”“快速审批”
- **Meta Description**：强调 1-2 小时审批、无需定金押金、免费申请
- **H1**：清晰描述主服务与地区定位
- **H2/H3**：覆盖服务类型、流程、FAQ
- **可读性**：使用列表、卡片、示例区块
- **首屏 CTA**：确保强吸引与快速转化

## EEAT 强化策略

- **E**（经验）：展示流程、示例、服务范围
- **E**（专业）：清楚解释审批、条款、审核逻辑
- **A**（权威）：显示合法合规与透明条款
- **T**（可信）：真实评价、24 小时支持、隐私说明
  > 建议补充：公司注册资料、执照编号、办公地址（如有）

## 关键技术点（Technical SEO）

- robots.txt 正确配置（已创建）
- sitemap.xml 正确配置（已创建）
- **canonical** 防止重复内容（需替换真实域名）
- 不设置 noindex
- 静态页面（已满足）
- 服务器稳定快速（CDN + 缓存）
- 状态码正常（200）
- 移动端优化（响应式布局）

## 可信度与转化设计

- 明确「无需定金押金」「免费申请」「合法合规」
- 表单字段简洁：名字、电话、地区、净收入、贷款数额
- 加强“隐私说明”和“清楚条款”提示
- 提供 WhatsApp 或电话入口

## 上线前必须替换

- `https://www.example.com/` 改为真实域名（`index.html`、`robots.txt`、`sitemap.xml`）
- 补充真实联系方式、公司信息与地址
- 如有执照编号请写入页尾增强可信度
